# MLP-proj4-Fall-2025
CAU Machine Learning Project Class - Term Project 4 Repository

# Descriptions
All files except `mlp-proj4-fall-2025-2-bonus.ipynb` must be executed in the Kaggle notebook and require the addition of the Hull Tactical Competition training data.
`mlp-proj4-fall-2025-2-bonus.ipynb` must be executed in the Kaggle notebook and require `kospi_dataset.csv` training data.

- `...baseline.ipynb` : baseline code (step 1)
- `...step2.ipynb` : step 2 code
- `...step3+4.ipynb` : step 3 and 4 code
- `...step5-1.ipynb` : step 5 code (additional features + CatBoost)
- `...step5-2.ipynb` : step 5 code (Enhanced CatBoost)
- `...step5-3.ipynb` : step 5 code (Ensembled Model)
- `...bonus.ipynb` : "Optional Bonus: Cross-Market Extension" Model code
- `kospi_dataset.csv` : dataset for "Optional Bonus: Cross-Market Extension"
- `...bonus-dataset.py` : code for acquiring `kospi_dataset.csv` dataset
